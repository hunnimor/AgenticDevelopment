## Skills

This folder contains reusable **execution skills** for AI Developer Agents.

Each skill is a focused playbook that can be applied across tasks and project types.

### Available Skills

- `architecture-decision-skill.md` - How to make architecture decisions with trade-off analysis.
- `feature-delivery-skill.md` - End-to-end implementation flow from task to tested feature.
- `quality-gate-skill.md` - Test, lint, review, and release readiness checks.

### How to Use

1. The orchestrator selects a skill based on task type.
2. The assigned agent follows the skill steps.
3. Outcomes are reported back in a structured format (decisions, risks, status).
